# ✨ 命运匹配 (Destiny Match)

> AI 预测你的未来伴侣，探索属于你的缘分

**命运匹配**是一款趣味性娱乐应用，利用先进的 AI 技术，为你生成一张"未来伴侣"的照片，并提供富有玄学色彩的般配度分析和缘分解读。

---

## 🎯 产品介绍

想知道未来的 TA 长什么样？和你们有多般配？

只需上传一张照片，AI 就能为你：
- 生成一张专属的未来伴侣照片
- 分析你们的般配度指数（60-98 分）
- 解读你们的缘分和性格互补

无论是好奇、娱乐还是社交分享，命运匹配都能为你带来有趣的体验。

> **⚠️ 免责声明**：本应用纯属娱乐性质，生成的伴侣照片与本人相貌无必然关联，不涉及真实的婚恋匹配服务。

---

## 🌟 功能特性

| 功能 | 描述 |
|------|------|
| 🎨 **AI 生成伴侣照片** | 基于即梦 AI，生成高质量的人物写真风格照片 |
| 💕 **多维度般配度分析** | 从情感共鸣、沟通风格、核心价值观等维度分析匹配度 |
| 🎭 **四种伴侣风格** | 温柔型、阳光型、知性型、神秘型，满足不同喜好 |
| 📤 **一键分享** | 生成精美的分享卡片，轻松分享到社交媒体 |
| 📜 **历史记录** | 自动保存最近 5 次生成记录，随时回顾 |
| 🔒 **隐私保护** | 照片仅用于本次生成，不会持久化存储 |

### 每日使用限制

为保证服务质量和公平使用，每位用户每日最多可生成 **3 次**。次数用完后需等待次日自动重置。

---

## 📱 使用流程

### 步骤 1：上传照片
点击或拖拽上传你的照片，支持 JPG、PNG、WEBP 格式（最大 5MB）。清晰的正面照效果会更好。

### 步骤 2：选择风格
选择你想要的伴侣风格，或让命运为你随机选择。

### 步骤 3：等待 AI 生成
等待约 15-30 秒，AI 正在为你生成专属结果...

### 步骤 4：查看结果并分享
查看生成的伴侣照片、般配度分数和缘分解读，一键分享给朋友。

## 🔒 隐私说明

我们高度重视用户隐私：

- ✅ **照片仅用于本次生成**，不会用于其他任何用途
- ✅ **照片不会持久化存储**，生成完成后立即删除临时文件
- ✅ **数据传输加密**，使用 HTTPS 保护传输安全
- ✅ **无需注册账号**，不收集个人身份信息

详细隐私政策请在应用内查看。

## 🚀 快速开始（开发者）

### 环境要求

- Node.js 18+
- npm 9+

### 安装与运行

```bash
# 1. 安装依赖
npm run install:all

# 2. 配置环境变量
cp server/.env.example server/.env
# 编辑 server/.env，填入 API 密钥

# 3. 启动开发服务器
npm run dev
```

应用将在以下地址运行：
- 前端：http://localhost:3000
- 后端：http://localhost:3001

### 环境变量

```bash
# server/.env
PORT=3001
CLIENT_URL=http://localhost:3000

# AI 服务 API 密钥（可选，未配置时使用 Mock 模式）
DREAMINA_API_KEY=your_key_here
SILICONFLOW_API_KEY=your_key_here
```
